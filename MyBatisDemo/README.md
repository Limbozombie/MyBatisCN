# [MyBatisDemo](https://github.com/yeecode/MyBatisDemo)

该项目是《通用源码阅读指导书——MyBatis源码详解》书籍的配套示例。

---

该项目包含29个示例，涉及MyBatis功能使用示例、MyBatis源码相关知识等。在使用该示例前，需先用`datasource`文件夹下的文件初始化两个数据表，并建立好数据库连接。
